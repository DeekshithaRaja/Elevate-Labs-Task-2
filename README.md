SHORT DESCRIPTION ABOUT THE PROJECT:

In this project, we developed a To-Do List web application using HTML, CSS, and JavaScript where users can add tasks through an input field and display them dynamically on the screen. We implemented functionality to mark tasks as completed and also provided an option to delete individual tasks. JavaScript was used to handle user interactions and update the UI instantly without reloading the page by manipulating the DOM. Overall, this project demonstrates how to build an interactive and dynamic web application using basic frontend technologies.
